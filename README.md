# Realistic Generators

Realistic steam (and, later, nuclear) power generation for Minecraft —
built for **All The Mods 10** and to interoperate with other mods through
tags + capabilities, never hard dependencies.

**Website:** https://realisticgenerators.benware.io

This repository hosts the **website** (GitHub Pages) and the **issue tracker**.

## Found a bug? Have a suggestion?

Open an [issue](../../issues) — a short bug-report form will guide you. Please
include your Minecraft / NeoForge / Realistic Generators versions and a log if
you have one.
